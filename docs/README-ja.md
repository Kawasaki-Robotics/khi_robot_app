# khi_robot_app

```khi_robot```のアプリケーションリポジトリ

## khi_robot_sample

MoveIt!を使った```khi_robot```のサンプルアプリケーション

duAro:
```roslaunch khi_robot_sample sample_duaro.launch```

## khi_duaro_onigiri_system

おにぎりの番重詰めを自動化したロボットシステムです。

MoveIt!を使用し**duAro**を制御します。
